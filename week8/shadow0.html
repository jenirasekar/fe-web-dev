CSS3 Box Shadows Explained

Master box shadows to add depth and dimension to web elements

What is Box Shadow?

The box-shadow property adds shadow effects around an element's frame. You can set multiple effects on one element, separated by commas.

The property values include:

  • offset-x: Horizontal shadow position (required)
  • offset-y: Vertical shadow position (required)
  • blur-radius: Blur distance (optional)
  • spread-radius: Shadow size (optional)
  • color: Shadow color (optional)
  • inset: Changes shadow to inner shadow (optional)
/* Basic syntax */ box-shadow: offset-x offset-y blur-radius spread-radius color inset; /* Example */ box-shadow: 5px 5px 10px 0 rgba(0,0,0,0.5);

Key Points

  • box-shadow can add one or more shadow effects to an element
  • Parameter order: horizontal offset → vertical offset → blur radius → spread radius → color
  • Use the inset keyword to create inner shadow effects
  • Multiple shadows are separated by commas, with earlier ones appearing on top
  • Shadows don't affect layout or trigger scrollbars
  • Proper use of shadows enhances UI depth and dimensionality
